# 🚀 Deploy Rápido na Vercel - CardioIA

## ⚡ Guia Expresso (5 minutos)

### 1️⃣ Acesse a Vercel
```
🔗 https://vercel.com
```

### 2️⃣ Faça Login
- Clique em **"Sign Up"**
- Escolha **"Continue with GitHub"**
- Autorize a Vercel

### 3️⃣ Importe o Projeto
- Clique em **"Add New..."** → **"Project"**
- Selecione o repositório **`cardio-app`** ou **`cardioia-app`**
- Clique em **"Import"**

### 4️⃣ Configure (Automático)
A Vercel detecta automaticamente:
```
Framework: Vite
Build Command: npm run build
Output Directory: dist
```

✅ **NÃO precisa alterar nada!**

### 5️⃣ Deploy
- Clique em **"Deploy"**
- Aguarde 2-3 minutos
- 🎉 **Pronto!**

---


## 🆘 Problemas?

### Build Failed?
```bash
npm run build
```
Se funcionar localmente, tente novamente na Vercel.

### Página em branco?
Verifique o console do navegador (F12)

### Mais ajuda?
Consulte o [Tutorial Completo](./TUTORIAL_DEPLOY_VERCEL.md) seção Troubleshooting.
